# Stability-First для ImageNet

Эксперимент по применению Stability-First подхода к ImageNet (или CIFAR-100 как замена для тестирования).

## Описание

Этот эксперимент адаптирует метод Stability-First для работы с ImageNet (224x224x3). Используется предобученный ResNet18 или упрощенная CNN архитектура.

**Примечание**: Полный ImageNet требует загрузки датасета вручную. Если ImageNet недоступен, скрипт автоматически использует CIFAR-100 для тестирования.

## Структура

- **Task A**: Первая половина классов (для ImageNet: 0-499, для CIFAR-100: 0-49)
- **Task B**: Вторая половина классов (для ImageNet: 500-999, для CIFAR-100: 50-99)

## Запуск

```bash
cd 08-stability-first-imagenet
pip install -r requirements.txt
python run_demo.py
```

## Требования

- Для полного ImageNet: загрузите датасет в `./data/ImageNet/`
- Для тестирования: CIFAR-100 будет загружен автоматически

## Ожидаемые результаты (CIFAR-100)

- **Task A before B**: ~60-70%
- **Task B after B (baseline)**: ~60-70%
- **Task A after B (baseline)**: ~0-10% (катастрофическое забывание)
- **Task A after B (stability-first)**: ~50-65% (сохранение знаний)

## Архитектура модели

- **ResNetBackbone**: Предобученный ResNet18 с новым head
- **SimpleImageNetCNN**: Упрощенная CNN (если нет предобученной модели)

## Примечания

- Из-за размера ImageNet, эксперимент может занять много времени
- Рекомендуется использовать GPU
- Для быстрого тестирования используется CIFAR-100
