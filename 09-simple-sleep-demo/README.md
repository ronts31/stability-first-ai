# 09. Simple Sleep Demo

## Описание

Простой демонстрационный эксперимент, показывающий концепцию "сна" для нейронных сетей. Модель оптимизируется на стабильность без использования реальных данных, что создает "широкий аттрактор" и повышает устойчивость к повреждениям весов.

## Ключевая идея

**Стабильность через сон**: Модель, которая "спала" (оптимизировалась только на стабильность к шуму), должна быть более устойчивой к повреждениям весов, чем модель, которая не спала.

## Архитектура

- **SimpleNet**: Простая полносвязная сеть с 2 скрытыми слоями
- **Stability Loss**: MSE между выходом на исходных данных и выходом на данных с шумом
- **Sleep Cycle**: Оптимизация только на стабильность без реальных данных

## Запуск

```bash
cd 09-simple-sleep-demo
pip install torch numpy matplotlib
python simple_sleep_demo.py
```

## Ожидаемый результат

После применения шума к весам, модель "Dreamer" (которая спала) должна показать более высокую точность, чем модель "Insomniac" (которая не спала).

## Технические детали

- **Входные данные**: Случайные векторы размерности 10
- **Задача**: Бинарная классификация (сумма элементов > 0)
- **Размер батча**: 64
- **Итерации сна**: 500
- **Уровень шума**: 0.2
